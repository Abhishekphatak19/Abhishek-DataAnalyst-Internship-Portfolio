# 📊 Data Analyst Internship Portfolio

## 👨‍💻 About Me
Hi, I'm Abhishek, a Data Analyst enthusiast with a strong foundation in data analysis, visualization, and business intelligence. This repository showcases my complete internship work using a real-world Online Retail dataset.

---

## 🎯 Internship Objective
To analyze retail data, uncover business insights, and build data-driven solutions using Python, SQL, and Power BI.

---

## 📁 Project Overview

This portfolio includes:

### 🔹 Part 1: Data Cleaning & Preparation
- Handled missing values and duplicates  
- Converted date columns  
- Created new features (order year, month, weekday)  
- Removed invalid data (negative quantity, price)

---

### 🔹 Part 2: Exploratory Data Analysis (EDA)
- Descriptive statistics  
- Univariate & multivariate analysis  
- Outlier detection (IQR, boxplots)  
- SQL queries for business insights  
- Static dashboard mock-up  

---

### 🔹 Part 3: Deep-Dive Analysis & Dashboard
- Defined KPIs (Revenue, Orders, AOV, Customers)  
- Customer Segmentation using RFM  
- Built interactive Power BI dashboard  
- Added filters, KPI cards, and visuals  

---

### 🔹 Part 4: Data Storytelling & Statistical Validation
- Created business narrative  
- Developed presentation deck  
- Performed hypothesis testing (T-test, correlation)  
- Generated business recommendations  

---

## 📊 Key KPIs

- 💰 Total Revenue  
- 📦 Total Orders  
- 👥 Total Customers  
- 🛒 Average Order Value (AOV)  

---

## 📈 Tools & Technologies Used

- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- SQL  
- Power BI  
- Excel  

---

## 🔍 Key Insights

- A small percentage of customers contribute most revenue  
- Sales are concentrated in specific countries  
- Certain products dominate revenue generation  
- Seasonal trends affect purchasing behavior  

---

## 💡 Business Recommendations

- Target high-value customers with loyalty programs  
- Re-engage inactive customers  
- Focus marketing on top-performing regions  
- Promote best-selling products
